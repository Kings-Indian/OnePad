# OnePad

**OnePad** is a custom macropad inspired by *One Piece*!

## Features

- 8 programmable keys  
- 1 rotary encoder  
- 1 OLED display

## Bill of Materials (BOM)

> For exact details, refer to `bom.csv`.

- 1× OLED display  
- 8× Keycaps  
- 8× Cherry MX switches  
- 8× SK6812 MINI-E LEDs  
- 9× Diodes  
- 1× Seeed Studio XIAO RP2040  
- 4× M3 5mm screws  
- 4× M3 8mm screws
